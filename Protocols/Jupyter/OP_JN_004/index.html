Transformation of plasmids in 96-well format (OT thermocycler version)

Description

This protocol carries out the transformation of a 96-well plate with chemically competent cells with plasmids in another 96-well plate, using the Opentrons Thermocycler module

For help with how to run Jupyter Notebook protocols, refer to our tutorial

Labware and Liquids

  • Left mount: An 8-channel p20 pipette
  • Right mount: An 8-channel p300 pipette
  • Slot 1: DNA plate (96-well PCR plate)
  • Slot 3 (don't place the plate here yet): Competent cell plate on temperature module (96-well PCR plate)
  • Slots 4 and 7: Tipracks for both pipettes
  • Slot 5: 12-well reservoir for LB

Execution

  • Set up the deck as described above
  • Run the cells one by one, paying attention to prompts and instructions

Tips and best practices

This protocol requires the jupyter notebook to have access to the thermocycler lid, which is not available from start. Follow the instructions at the start of the protocol to set it up

It might be usefulk having a bucket of ice to move the competent cell plate to/from the robot deck
