Plating of cells from 96-well plates

Description

This protocol carries out plating of cells in a 96-well plate to multiple 6-well plates with solid culture media

For help with how to run Jupyter Notebook protocols, refer to our tutorial

Labware and Liquids

  • Right mount: A p300 pipette
  • Slot 10: A 96-well PCR plate with cells
  • Slots 1 to 9: 6-well plates up to the necessary number
  • Slot 11: Tiprack for the pipette

Execution

  • Label the required number of plates appropriately. Simulate the protocol first to see how the plating sequence will happen (A1 through H1, then A2 through H2, etc)
  • Set up the deck as described above
  • Run the cells one by one, paying attention to prompts and instructions

Script

Plating of cells from 96-well plates

Tips and best practices

Depending on the required number of samples, this protocol requires you to change the plates on the deck after the first 9 are done

To avoid contamination from the tips going to the trash bin, it's advisable to start with all the plates covered and remove their lids as the protocol progresses

Move plates to a nutator at slow rotation speed after this protocol is done, so that the cells can be spreaded properly
