PCR Purification Using Magnetic Beads

Description

This protocol purifies a rack of 96 PCR samples utilising magnetic beads and the magnetic module.

Labware and Liquids

  • Left mount: An 8-channel p20 pipette (GEN2)
  • Right mount: An 8-channel p300 pipette (GEN2)
  • Slot 1: Reservoir to collect liquid waste
  • Slot 4: Magnetic module (GEN2) with 96 well PCR plate containing 20µl of PCR product
  • Slot 7: Twelve channel reservoir containing beads, ethanol wash and elution buffer
  • Slots 2, 3, 5, 6, 8, and 9: 300µl Tip Racks
  • Slot 11: 20µl Tip Rack

Execution

  • Set up the deck as described above
  • Run the protocol
  • The protocol contains pauses and manual steps to empty the waste box and check the beads.

Script

PCR Purification Using Magnetic Beads

Tips and best practices

Carry out a test run of the protocol using the labware and water to ensure that the positioning of labware and tips is suitable. Re-calibration of the deck or labware may be required.

Ensure the beads are fully dry prior to elution to prevent any ethanol carry-over into the final elution.
